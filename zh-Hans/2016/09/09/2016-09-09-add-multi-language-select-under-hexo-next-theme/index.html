Add Multi-Language select under Hexo NexT theme

Welcome to My Blog !

I want to write blog in both English and Chinese. Then more developers can access them. So I add one selection box on header.

Following are the steps:

Installation

*Install hexo-generator-i18n module.

Config the module.

  • _config.yml

    • Modify the themes/next/_config.yml

    • config hexo-generator-i18n option (optional, this is default option)

Replace the url_for() to url_for_lang() to output link under current language.

  • Most the url_for() functions under /themes/next/_macro/
  • *Note: If the sources(such avatar.png) are under the same folder, you will still use the url_for().
